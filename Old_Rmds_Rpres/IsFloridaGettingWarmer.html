Is Florida Getting Warmer?

Here we look at statewide temperatures in Florida.

Get the data into R.

L = 'http://myweb.fsu.edu/jelsner/data/FLMonthlyT.txt'
df = read.table(L, header = TRUE)

L is a string locating the data sheet (FLMonthlyT.txt) on a server. The data is read into R using the read.table function. The first argument to the function is L. The result is a data frame called df. You can use any name you want.

You can click on the df in the Environment panel to bring up the data sheet.

Make a plot of the data.

library("ggplot2")
ggplot(df, aes(x = Year, y = Annual)) +
  geom_line() + ylab("Annual Temperature (F)")
